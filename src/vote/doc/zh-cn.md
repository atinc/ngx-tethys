<div class="dg-alert dg-alert-info">用于点赞和收藏等的投票组件。</div>

## 何时使用

对事物进行快速点击交互反馈，且需要对总量进行展示

## 模块导入

``` ts
import { ThyVoteModule } from 'ngx-tethys/vote';
```

## 基本使用

``` ts
<thy-vote [thyVoteCount]="5" [thyHasVoted]="true"></thy-vote>
```
