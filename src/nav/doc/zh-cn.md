## 何时使用

用户需要使用导航在各个页面、各个部分中进行跳转，全局导航会提供全局性的类目和功能，局部导航会收纳和排列网站架构。

## 模块导入
```ts
import { ThyNavModule } from "ngx-tethys/nav";
```
