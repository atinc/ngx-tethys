<div class="dg-alert dg-alert-info">简单的文字提示框。</div>

## 何时使用
- 鼠标移入时显示提示，移出消失。
- 用来代替系统默认的`title`提示，提供一个`文字/按钮/操作`的文案解释。

## 模块导入
```ts
import { ThyTooltipModule } from "ngx-tethys/tooltip";
```
