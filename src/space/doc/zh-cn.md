<alert>设置间距。</alert>

## 何时使用
当内部元素需要一段距离时使用，避免紧紧挨在一起。

## 模块导入
```ts
import { ThySpaceModule } from "ngx-tethys/space";
```

## 大小
Space 支持大小`'zero' | 'xxs' | 'xs' | 'sm' | 'md' | 'lg' | 'xlg'`，同时支持自定义数字。分别为 0px, 4px, 8px, 12px, 16px, 20px, 24px, 28px。
