<alert>内容分割线。</alert>

## 何时使用
当需要对不同段落的文本进行分割，对行内文字/链接进行分割时使用。

## 模块导入
```ts
import { ThyDividerModule } from "ngx-tethys/divider";
```

## 基本使用

```html
<thy-divider [thyStyle]="'solid'"></thy-divider>
```
