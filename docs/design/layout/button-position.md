## 表单/表格/列表

- 常规显示，当前使用频率较多的按钮位置
- 表单、表格、列表等规则操作中，“确定”“取消”位于模态窗的右下角

## 模态窗提示/气泡确认框

- 当模态窗作为提示功能，仅有文字时，此时“确定”“取消”位于模态窗的右下角

## 模态窗操作

- 存在于比较复杂的模态窗操作中，通常有字段选择、成员选择、全局关联等，此时“确定”“取消”位于模态窗的右下角
